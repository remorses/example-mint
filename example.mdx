Holocron supports MDX, MDX is a superset of markdown, it allows you to embed React components in your markdown

Try creating an MDX CardGroup component using the slash command, type / and cardgroup

<HlcCardGroup cols={2}>
    <HlcCard icon="" title="Card 1" href="">
        Description 1
    </HlcCard>
    <HlcCard icon="" title="Card 2" href="">
        Description 2
    </HlcCard>
    <HlcCard icon="" title="Card 3" href="">
        Description 3
    </HlcCard>
    <HlcCard icon="" title="Card 4" href="">
        Description 4
    </HlcCard>
</HlcCardGroup>
